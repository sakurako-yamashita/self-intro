動画のURL・タイトル・概要欄などを貼り付けて「分析実行」。Gemini が4項目を抽出し、左列に表示します。右列は手入力で自分の構想案と並べて比較できます（ページをリロードせず更新）。

入力

動かし方： APIキーはブラウザに置きません。このHTMLと同じプロジェクトを Vercel にデプロイし、環境変数 GEMINI_API_KEY を設定してください。 GitHub Pages だけの場合は、上の「API のベースURL」に https://あなたのプロジェクト.vercel.app を入れ、末尾スラッシュなしで保存してから分析を実行してください。
